## 项目介绍

基于springboot的Echarts管理系统：一个echarts使用的实例系统、可以在网页上添加自己的图表、拖动图表位置进行布局

## 主要功能：

- 图表管理：添加图表、图表列表展示、图表查看、图表信息编辑、图表删除
- 图表布局：选框添加图表、拖动图表、删除布局中的图表
- 操作日志：条件查询、日志列表、删除日志、导出日志、查看日志详情
